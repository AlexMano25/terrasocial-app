## Introduction

Cher Maitre Jean-Noel,

Bienvenue sur **TERRASOCIAL**, la plateforme de gestion du foncier social. Ce guide a ete concu pour vous accompagner, pas a pas, dans l'utilisation du tableau de bord juridique mis a la disposition de votre cabinet.

En tant que partenaire juridique de TERRASOCIAL, vous disposez d'un espace dedie vous permettant de gerer les dossiers d'inscription fonciere de vos clients, d'echanger des documents juridiques, de communiquer avec les parties prenantes et de suivre la facturation de vos prestations.

Ce guide couvre l'ensemble des **7 panneaux** de votre tableau de bord. Nous vous recommandons de le lire integralement avant votre premiere utilisation.

## 1. Connexion a la plateforme

### Acces au portail

1. Ouvrez votre navigateur web (Chrome, Firefox ou Safari recommandes).
2. Rendez-vous a l'adresse suivante :

   **https://social.manovende.com/login.html**

3. Saisissez votre **adresse e-mail professionnelle** dans le champ prevu a cet effet.
4. Saisissez votre **mot de passe** fourni lors de la creation de votre compte partenaire.
5. Cliquez sur le bouton **"Se connecter"**.

[Capture d'ecran : Page de connexion avec les champs e-mail et mot de passe]

### Premiere connexion

Lors de votre premiere connexion, il vous sera demande de :

- **Modifier votre mot de passe temporaire** par un mot de passe personnel (minimum 8 caracteres, incluant une majuscule, un chiffre et un caractere special).
- **Verifier les informations de votre cabinet** dans le panneau Profil.

### En cas d'oubli du mot de passe

1. Sur la page de connexion, cliquez sur **"Mot de passe oublie ?"**.
2. Renseignez votre adresse e-mail professionnelle.
3. Consultez votre boite de reception pour le lien de reinitialisation.
4. Suivez les instructions du courriel pour definir un nouveau mot de passe.

[Capture d'ecran : Lien "Mot de passe oublie" sur la page de connexion]

> **Remarque :** Le lien de reinitialisation expire au bout de 24 heures. Passe ce delai, vous devrez effectuer une nouvelle demande.

## 2. Panneau 1 - Tableau de bord

Le panneau **Tableau de bord** constitue votre vue d'ensemble. Il est la premiere page qui s'affiche apres votre connexion. Il vous offre une vision synthetique de l'activite de votre cabinet sur la plateforme TERRASOCIAL.

[Capture d'ecran : Vue d'ensemble du tableau de bord avec les 6 KPI affiches]

### Les 6 indicateurs cles de performance (KPI)

Votre tableau de bord affiche **6 KPI** essentiels :

| N. | KPI | Description |
|----|-----|-------------|
| 1 | **Dossiers en attente** | Nombre de dossiers d'inscription necessitant votre revue juridique |
| 2 | **Dossiers valides** | Nombre total de dossiers que vous avez approuves |
| 3 | **Dossiers rejetes** | Nombre de dossiers que vous avez rejetes avec motif |
| 4 | **Documents en cours** | Nombre de documents (contrats, actes) en attente de traitement |
| 5 | **Messages non lus** | Nombre de messages non lus dans votre messagerie |
| 6 | **Factures en attente** | Montant total des factures en attente de paiement |

### Comment lire le tableau de bord

- Chaque KPI est affiche sous forme de **carte** avec une valeur numerique mise en evidence.
- Les cartes utilisent un **code couleur** pour signaler les elements urgents :
  - **Rouge** : action immediate requise (ex. dossiers en attente depuis plus de 48h)
  - **Orange** : attention necessaire (ex. documents en attente de signature)
  - **Vert** : situation normale, aucune action urgente

[Capture d'ecran : Detail d'une carte KPI avec le code couleur]

### Vue d'ensemble des dossiers

En dessous des KPI, vous trouverez un **tableau recapitulatif des dossiers recents** comprenant :

- Le numero de reference du dossier
- Le nom du client (souscripteur)
- Le type de dossier (inscription, transfert, modification)
- Le statut actuel (en attente, en cours, valide, rejete)
- La date de soumission

> **Astuce :** Cliquez sur un numero de dossier pour acceder directement a sa fiche complete dans le panneau Dossiers.

## 3. Panneau 2 - Dossiers

Le panneau **Dossiers** est le coeur de votre activite juridique sur TERRASOCIAL. C'est ici que vous examinez les inscriptions foncieres, emettez vos avis juridiques et decidez de la validation ou du rejet des demandes.

[Capture d'ecran : Vue du panneau Dossiers avec la liste des dossiers]

### 3.1 Acceder aux dossiers

1. Dans le menu lateral gauche, cliquez sur **"Dossiers"**.
2. La liste de tous les dossiers assignes a votre cabinet s'affiche.

### 3.2 Filtrer les dossiers

Pour retrouver rapidement un dossier, utilisez les filtres disponibles en haut de la liste :

**Filtrage par statut :**
- **En attente** : dossiers necessitant votre premiere revue
- **En cours** : dossiers en cours de traitement par votre cabinet
- **Valide** : dossiers approuves par vos soins
- **Rejete** : dossiers que vous avez refuses

**Filtrage par type :**
- **Inscription** : nouvelle demande d'inscription fonciere
- **Transfert** : demande de transfert de propriete
- **Modification** : modification d'une inscription existante

[Capture d'ecran : Barre de filtres avec les menus deroulants Statut et Type]

**Pour appliquer un filtre :**

1. Cliquez sur le menu deroulant **"Statut"** et selectionnez le statut souhaite.
2. Cliquez sur le menu deroulant **"Type"** et selectionnez le type de dossier.
3. Les resultats se mettent a jour automatiquement.
4. Pour reinitialiser les filtres, cliquez sur **"Effacer les filtres"**.

### 3.3 Examiner un dossier

1. Cliquez sur la **ligne du dossier** que vous souhaitez examiner.
2. La fiche complete du dossier s'ouvre, comprenant :
   - Les informations du souscripteur (nom, prenom, contact)
   - La description du terrain (localisation, superficie, references cadastrales)
   - Les pieces justificatives jointes par le client
   - L'historique des actions effectuees sur le dossier

[Capture d'ecran : Fiche complete d'un dossier avec les informations du souscripteur et du terrain]

### 3.4 Emettre un avis juridique

1. Dans la fiche du dossier, faites defiler jusqu'a la section **"Avis juridique"**.
2. Cliquez sur le bouton **"Rediger un avis"**.
3. Redigez votre analyse juridique dans le champ de texte prevu.
4. Vous pouvez joindre des documents complementaires en cliquant sur **"Ajouter une piece jointe"**.
5. Cliquez sur **"Enregistrer l'avis"** pour sauvegarder votre analyse.

[Capture d'ecran : Formulaire de redaction de l'avis juridique]

> **Important :** L'avis juridique est visible par l'ensemble des parties concernees par le dossier. Veillez a utiliser un langage professionnel et precis.

### 3.5 Valider un dossier

1. Apres avoir examine le dossier et redige votre avis, cliquez sur le bouton **"Valider"** (icone de coche verte).
2. Une fenetre de confirmation apparait.
3. Ajoutez un **commentaire de validation** si necessaire.
4. Cliquez sur **"Confirmer la validation"**.
5. Le statut du dossier passe automatiquement a **"Valide"**.

[Capture d'ecran : Fenetre de confirmation de validation d'un dossier]

### 3.6 Rejeter un dossier

1. Si le dossier presente des irregularites, cliquez sur le bouton **"Rejeter"** (icone de croix rouge).
2. Une fenetre s'ouvre vous demandant de **preciser le motif du rejet**.
3. Selectionnez un motif dans la liste predeterminee ou redigez un motif personnalise.
4. Cliquez sur **"Confirmer le rejet"**.
5. Le client sera automatiquement notifie du rejet et du motif.

[Capture d'ecran : Fenetre de rejet avec le champ de motif obligatoire]

> **Remarque :** Un dossier rejete peut etre resoumis par le client apres correction. Dans ce cas, il apparaitra de nouveau dans vos dossiers en attente.

### 3.7 Assigner un dossier a un collaborateur

Si votre cabinet dispose de collaborateurs enregistres sur la plateforme :

1. Ouvrez la fiche du dossier concerne.
2. Cliquez sur le bouton **"Assigner"** situe en haut a droite.
3. Selectionnez le **collaborateur** dans la liste deroulante.
4. Ajoutez des **instructions specifiques** si necessaire.
5. Cliquez sur **"Confirmer l'assignation"**.

Le collaborateur recevra une notification et le dossier apparaitra dans son propre tableau de bord.

[Capture d'ecran : Menu d'assignation d'un dossier a un collaborateur]

## 4. Panneau 3 - Documents

Le panneau **Documents** vous permet de gerer l'ensemble de vos modeles de contrats, d'envoyer des documents aux clients et de suivre les versions de chaque document.

[Capture d'ecran : Vue du panneau Documents avec la bibliotheque de modeles]

### 4.1 Acceder a la bibliotheque de documents

1. Dans le menu lateral gauche, cliquez sur **"Documents"**.
2. Vous accedez a la bibliotheque de documents organisee en categories :
   - **Modeles de contrats** : contrats types fournis ou crees par votre cabinet
   - **Documents clients** : documents recus ou envoyes a vos clients
   - **Archives** : documents des dossiers clotures

### 4.2 Uploader un modele de contrat

1. Cliquez sur le bouton **"Nouveau modele"** en haut a droite.
2. Dans la fenetre qui s'ouvre :
   - Saisissez le **nom du modele** (ex. : "Contrat de cession fonciere - Standard").
   - Selectionnez la **categorie** du document.
   - Cliquez sur **"Parcourir"** pour selectionner le fichier depuis votre ordinateur.
   - Formats acceptes : PDF, DOCX, DOC.
   - Taille maximale : 10 Mo par fichier.
3. Cliquez sur **"Telecharger"** pour enregistrer le modele.

[Capture d'ecran : Formulaire d'upload d'un nouveau modele de contrat]

> **Astuce :** Nommez vos modeles de maniere descriptive pour les retrouver facilement (ex. : "Bail_Emphyteotique_V2_2026").

### 4.3 Envoyer un document a un client

1. Dans la liste des documents, identifiez le document a envoyer.
2. Cliquez sur l'icone **"Envoyer"** (icone d'avion en papier) a droite de la ligne.
3. Dans la fenetre d'envoi :
   - Selectionnez le **destinataire** (client souscripteur) via le menu deroulant ou en saisissant son nom.
   - Associez le document a un **dossier existant** si applicable.
   - Ajoutez un **message d'accompagnement** si necessaire.
4. Cliquez sur **"Envoyer"**.
5. Le client recevra une notification et pourra consulter le document depuis son propre espace.

[Capture d'ecran : Fenetre d'envoi de document a un client avec selection du destinataire]

### 4.4 Gestion des versions

Chaque document dispose d'un **historique de versions** :

1. Cliquez sur le document pour ouvrir sa fiche.
2. Faites defiler jusqu'a la section **"Historique des versions"**.
3. Vous y verrez :
   - Le numero de version (V1, V2, V3...)
   - La date de modification
   - L'auteur de la modification
   - Un lien de telechargement pour chaque version

**Pour ajouter une nouvelle version :**

1. Cliquez sur **"Ajouter une version"**.
2. Selectionnez le fichier mis a jour depuis votre ordinateur.
3. Ajoutez un **commentaire de revision** (ex. : "Mise a jour clause 4.2 - Conditions de resiliation").
4. Cliquez sur **"Enregistrer"**.

[Capture d'ecran : Historique des versions d'un document avec les numeros de version]

### 4.5 Approuver ou rejeter un document

Lorsqu'un document est soumis pour approbation :

1. Le document apparait avec le badge **"En attente d'approbation"**.
2. Cliquez sur le document pour l'ouvrir et l'examiner.
3. Apres examen :
   - Cliquez sur **"Approuver"** (bouton vert) si le document est conforme.
   - Cliquez sur **"Rejeter"** (bouton rouge) en precisant le motif si des corrections sont necessaires.

[Capture d'ecran : Boutons Approuver et Rejeter sur la fiche d'un document]

## 5. Panneau 4 - Messagerie

Le panneau **Messagerie** vous permet de communiquer directement avec les clients, les proprietaires fonciers et les autres services juridiques impliques dans vos dossiers.

[Capture d'ecran : Vue du panneau Messagerie avec la liste des conversations]

### 5.1 Acceder a la messagerie

1. Dans le menu lateral gauche, cliquez sur **"Messagerie"**.
2. La liste de vos conversations s'affiche, classees par date du dernier message.
3. Les conversations non lues sont signalees par un **indicateur bleu**.

### 5.2 Lire et repondre a un message

1. Cliquez sur une **conversation** dans la liste pour l'ouvrir.
2. L'historique complet des echanges s'affiche dans l'ordre chronologique.
3. Pour repondre :
   - Saisissez votre message dans le champ de texte en bas de la fenetre.
   - Cliquez sur **"Envoyer"** ou appuyez sur la touche **Entree**.

[Capture d'ecran : Fenetre de conversation avec l'historique des messages]

### 5.3 Demarrer une nouvelle conversation

1. Cliquez sur le bouton **"Nouveau message"** (icone de crayon) en haut a droite.
2. Dans le champ **"Destinataire"**, saisissez le nom ou l'adresse du contact :
   - **Client** : saisissez le nom du souscripteur
   - **Proprietaire** : saisissez le nom du proprietaire foncier
   - **Service juridique** : selectionnez le service dans la liste
3. Redigez l'**objet** de votre message.
4. Redigez le **corps** du message.
5. Cliquez sur **"Envoyer"**.

[Capture d'ecran : Formulaire de nouveau message avec les champs destinataire et objet]

### 5.4 Joindre des pieces

1. Dans la fenetre de conversation, cliquez sur l'icone **"Piece jointe"** (icone de trombone).
2. Selectionnez le fichier depuis votre ordinateur.
3. Formats acceptes : PDF, DOCX, JPG, PNG.
4. Taille maximale : 5 Mo par piece jointe.
5. Le fichier apparait en miniature dans le champ de saisie.
6. Cliquez sur **"Envoyer"** pour transmettre le message avec la piece jointe.

[Capture d'ecran : Icone de piece jointe et apercu du fichier avant envoi]

### 5.5 Types d'interlocuteurs

Votre messagerie vous permet d'echanger avec trois categories d'interlocuteurs :

| Interlocuteur | Cas d'utilisation |
|---|---|
| **Clients (souscripteurs)** | Demande de complements d'information, notification de l'avancement du dossier, envoi de documents |
| **Proprietaires fonciers** | Verification des titres, demande de pieces complementaires, coordination des signatures |
| **Services juridiques** | Consultation inter-cabinets, arbitrage, coordination sur des dossiers communs |

## 6. Panneau 5 - Facturation

Le panneau **Facturation** vous permet de suivre les revenus generes par les souscriptions actives et de gerer les demandes de paiement.

[Capture d'ecran : Vue du panneau Facturation avec le recapitulatif financier]

### 6.1 Comprendre le modele de facturation

Le modele de facturation de TERRASOCIAL est simple et transparent :

- **Tarif :** 100 FCFA par jour par souscription active
- **Calcul :** Le montant est calcule automatiquement en fonction du nombre de jours d'activite de chaque souscription
- **Cycle de facturation :** Mensuel (du 1er au dernier jour du mois)

**Exemple :** Si votre cabinet gere 50 souscriptions actives pendant un mois de 30 jours, le montant facturable est de :
50 souscriptions x 100 FCFA x 30 jours = **150 000 FCFA**

### 6.2 Consulter le recapitulatif financier

1. Dans le menu lateral gauche, cliquez sur **"Facturation"**.
2. Le recapitulatif affiche :
   - **Souscriptions actives** : nombre de souscriptions actuellement en cours
   - **Montant du mois en cours** : total provisoire calcule a la date du jour
   - **Montant du mois precedent** : total facture le mois dernier
   - **Historique des paiements** : liste des paiements recus

[Capture d'ecran : Recapitulatif financier avec les montants mensuels]

### 6.3 Generer une facture

1. Cliquez sur le bouton **"Generer une facture"**.
2. Selectionnez la **periode de facturation** (mois concerne).
3. Verifiez le **detail des souscriptions** et des montants.
4. Cliquez sur **"Generer"**.
5. La facture est creee au format PDF et apparait dans la liste des factures.

[Capture d'ecran : Formulaire de generation de facture avec selection de la periode]

### 6.4 Demander un paiement

1. Dans la liste des factures, identifiez la facture concernee.
2. Cliquez sur le bouton **"Demander le paiement"** (icone de billet).
3. Une notification de demande de paiement est envoyee a l'administrateur TERRASOCIAL.
4. Le statut de la facture passe a **"Paiement demande"**.
5. Une fois le paiement effectue, le statut passera a **"Paye"**.

[Capture d'ecran : Bouton de demande de paiement sur une ligne de facture]

### 6.5 Statuts des factures

| Statut | Signification |
|---|---|
| **Brouillon** | Facture generee mais non soumise |
| **Soumise** | Facture envoyee pour traitement |
| **Paiement demande** | Demande de paiement en cours |
| **Paye** | Paiement recu et confirme |
| **En retard** | Paiement non recu dans les delais |

## 7. Panneau 6 - Collaborateurs

Le panneau **Collaborateurs** vous permet de gerer les membres de votre cabinet ayant acces a la plateforme TERRASOCIAL.

[Capture d'ecran : Vue du panneau Collaborateurs avec la liste des membres du cabinet]

### 7.1 Pourquoi ajouter des collaborateurs ?

En tant que responsable de votre cabinet, vous pouvez inviter vos collegues et associes a utiliser la plateforme. Chaque collaborateur peut se voir attribuer un niveau de droits adapte a ses responsabilites.

### 7.2 Creer un compte collaborateur

1. Dans le menu lateral gauche, cliquez sur **"Collaborateurs"**.
2. Cliquez sur le bouton **"Ajouter un collaborateur"** en haut a droite.
3. Renseignez les informations suivantes :
   - **Nom complet** du collaborateur
   - **Adresse e-mail professionnelle** (elle servira d'identifiant de connexion)
   - **Poste / Fonction** au sein du cabinet
   - **Niveau de droits** (voir section 7.3)
4. Cliquez sur **"Creer le compte"**.
5. Le collaborateur recevra un courriel d'invitation avec un mot de passe temporaire.

[Capture d'ecran : Formulaire de creation d'un compte collaborateur]

### 7.3 Gestion des droits d'acces

Trois niveaux de droits sont disponibles :

| Niveau | Permissions |
|---|---|
| **Lecture** | Consultation des dossiers, documents et messages. Aucune modification possible. Ideal pour les stagiaires ou les consultants externes. |
| **Ecriture** | Toutes les permissions de lecture, plus : redaction d'avis juridiques, envoi de documents, reponse aux messages, validation/rejet de dossiers. Ideal pour les avocats collaborateurs. |
| **Admin** | Toutes les permissions d'ecriture, plus : creation/suppression de collaborateurs, modification des parametres du cabinet, acces a la facturation. Reserve au responsable du cabinet. |

**Pour modifier les droits d'un collaborateur :**

1. Dans la liste des collaborateurs, cliquez sur le **nom** du collaborateur concerne.
2. Dans sa fiche, cliquez sur le menu deroulant **"Niveau de droits"**.
3. Selectionnez le nouveau niveau.
4. Cliquez sur **"Enregistrer les modifications"**.

[Capture d'ecran : Modification du niveau de droits d'un collaborateur]

### 7.4 Activer ou desactiver un collaborateur

Pour suspendre temporairement l'acces d'un collaborateur sans supprimer son compte :

1. Dans la liste des collaborateurs, identifiez la personne concernee.
2. Cliquez sur le **bouton bascule** dans la colonne "Statut" :
   - **Actif** (vert) : le collaborateur peut se connecter et utiliser la plateforme.
   - **Inactif** (gris) : le collaborateur ne peut plus se connecter. Ses donnees et son historique sont conserves.
3. La modification prend effet immediatement.

[Capture d'ecran : Bouton bascule Actif/Inactif pour un collaborateur]

> **Remarque :** Un collaborateur desactive conserve son compte. Vous pouvez le reactiver a tout moment sans perte de donnees.

## 9. Conseils pour une utilisation efficace

### Organisation quotidienne

- **Consultez votre tableau de bord chaque matin** pour identifier les dossiers urgents et les messages non lus.
- **Traitez les dossiers en attente dans un delai de 48 heures** pour maintenir la satisfaction de vos clients et respecter les engagements de la plateforme.
- **Utilisez les filtres** du panneau Dossiers pour prioriser votre travail par statut ou par type.

### Gestion documentaire

- **Normalisez le nommage de vos modeles** de contrats (ex. : Type_Description_Version_Annee).
- **Mettez a jour vos modeles regulierement** et utilisez le systeme de versions pour garder une trace des modifications.
- **Archivez les documents des dossiers clotures** pour desencombrer votre bibliotheque active.

### Communication

- **Repondez aux messages dans les 24 heures** pour maintenir une bonne relation avec vos clients et les autres parties.
- **Utilisez les pieces jointes** pour transmettre les documents directement via la messagerie plutot que par des canaux externes.
- **Preciser le numero de dossier** dans l'objet de vos messages pour faciliter le suivi.

### Facturation

- **Generez vos factures en debut de mois** pour le mois precedent afin de maintenir une tresorerie reguliere.
- **Verifiez le nombre de souscriptions actives** avant de generer une facture pour vous assurer de l'exactitude des montants.

### Securite

- **Ne partagez jamais vos identifiants** de connexion avec des tiers.
- **Desactivez les comptes** des collaborateurs ayant quitte le cabinet.
- **Changez votre mot de passe** tous les 3 mois.
- **Deconnectez-vous** systematiquement apres chaque session, surtout sur un ordinateur partage.
