================================================================================
TROUBLESHOOTING
================================================================================

"?" button doesn't appear:
  • Check script loaded: window.helpSystem exists?
  • Check DOM ready: document.body exists?
  • Check console for JS errors

Modal doesn't open:
  • Check modal container exists: document.getElementById('modalContainer')
  • Check console for errors
  • Clear cache (Ctrl+Shift+R)

Content not showing:
  • Check JSON syntax (JSONLint.com)
  • Check ID placement (right role?)
  • Check duplicate IDs

Search not working:
  • Content must be in current context
  • Search queries must be 2+ characters
  • Check browser console for errors

Videos don't play:
  • Check URL is valid and CORS-enabled
  • Check domain is trusted
  • Consider using YouTube embeds
