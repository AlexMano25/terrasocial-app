KEY FUNCTIONS
================================================================================

showHelpModal()
  - Open help modal with current context
  - Usage: window.helpSystem.showHelpModal()

getCurrentContext()
  - Get current user role context
  - Returns: 'public' | 'client' | 'agent' | 'admin'
  - Usage: window.helpSystem.getCurrentContext()

addCustomHelp(context, sections)
  - Add custom help sections at runtime
  - Usage: window.helpSystem.addCustomHelp('client', [...])

setLanguage(lang)
  - Set language for localization (future)
  - Usage: window.helpSystem.setLanguage('en')

_searchHelpContent(query)
  - Search all help content (internal)
  - Returns: Array of matching results

_generateHelpModalHTML()
  - Generate modal HTML (internal)
  - Called when modal first opens

_injectModalStyles()
  - Inject modal styles (internal)
  - Called once per session
